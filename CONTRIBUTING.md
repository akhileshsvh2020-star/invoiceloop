# Contributing

Thanks for improving InvoiceLoop.

## Local Setup

```bash
pnpm install
cp .env.example .env
pnpm dev
```

## Branches And Commits

- Use short branch names such as `feat/invoice-crud` or `fix/mobile-table`.
- Use conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`, `test:`.
- Keep commits focused and reviewable.

## Before Opening A PR

```bash
pnpm lint
pnpm build
```

Add screenshots for UI changes and describe what changed plus why.
